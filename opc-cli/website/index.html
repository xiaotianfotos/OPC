System Architecture

模块化架构设计

从语音合成到语音识别，再到视频剪辑的完整工作流

01
🔊

TTS 语音合成

支持 Edge-TTS（微软在线）和 Qwen3-TTS（本地模型）双引擎。 内置 9 种音色，支持情绪指令、声音设计、声音克隆。

opc tts "Hello" -e qwen --speaker Vivian
02
🎙️

ASR 语音识别

基于 Qwen3-ASR-1.7B 和强制对齐模型， 4 阶段 Pipeline 生成 SRT/ASS 卡拉 OK 字幕。

opc asr audio.mp3 --format srt
03
✂️

Cut 视频剪辑

基于 ASR 字词级时间戳的可视化剪辑界面。 双击删除字词，拖拽选择，实时预览。

opc cut --video clip.mp4
04
📡

设备播放

自动发现 AirPlay 和 DLNA 设备， 支持 HomePod、Apple TV 等无线播放。

opc say "通知" --device HomePod
Core Features

核心功能矩阵

覆盖语音处理全流程的命令行工具

TTS

多引擎语音合成

Edge-TTS 提供 322 种在线音色，Qwen3-TTS 支持本地部署。 支持语速、音调、音量调节，以及情绪指令控制。

opc tts "你好" -e qwen --speaker Vivian --instruct "用温柔的语气"
ASR

4阶段 Pipeline

ASR + 强制对齐 → 断句 → CSV 修正 → 渲染输出。 每阶段保存中间文件，支持从任意阶段恢复执行。

opc asr audio.mp3 --format srt --fix-dir ./fixes
CUT

字词级视频剪辑

Web 可视化界面，基于 ASR 字词时间戳精确剪辑。 双击删除字词，Delete 键批量删除，Ctrl+A 全选。

opc cut --video /path/to/video.mp4 --port 9090
DEVICE

智能设备发现

自动扫描局域网内的 AirPlay 和 DLNA 设备。 支持 HomePod、Apple TV、智能电视等无线音频投放。

opc discover --set-default
opc say "播放完成" --device "Living Room"
ASR-SPLIT

字幕行拆分

当字幕行超过最大字符数时，按文本匹配精确拆分。 支持单条拆分和 CSV 批量处理，保留卡拉 OK 时间戳。

opc asr-split audio.lines.json --line 10 --after "理解，"
VOICE

声音克隆与设计

Qwen3-TTS 支持 voice_clone 模式克隆任意声音， voice_design 模式通过文本描述创建虚拟音色。

opc tts "你好" -e qwen --mode voice_clone --ref-audio ref.wav

Stage 1: ASR + 强制对齐

使用 Qwen3-ASR-1.7B 模型进行语音识别，结合 Qwen3-ForcedAligner 进行字词级时间戳对齐。

这是 GPU 最密集的阶段，输出包含每个字的精确开始/结束时间的 raw.json 文件。

→ 输出: audio.raw.json (GPU 密集，可复用)
